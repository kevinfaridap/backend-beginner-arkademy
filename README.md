* [Database](database-example.sql)


## Installation

Clone this repository and then use the package manager npm to install dependencies.


```bash
npm install
```

## Setup .env example

Create .env file in your root project folder.

```env

DB_HOST = localhost
DB_ USER = root
DB = yourdatabase

```

## Run the app

Development mode

```bash
npm run dev
```

Deploy mode

```bash
npm start
```
